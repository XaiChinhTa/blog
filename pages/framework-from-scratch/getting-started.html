Before we started

Init project and install Typescript, Typescript Node, Typescript definitions for Express(@types/express) and Express

    cd ../your-project-dir
    npm init
    npm install -s @types/express express

Install nodemon for monitoring any changes in your application(auto restart the server when any code changes). This step is optional

    npm install -s nodemon

Configure the Typescript configuration file

    {
        "compilerOptions": {
            "module": "commonjs",
            "moduleResolution": "node",
            "pretty": true,
            "sourceMap": true,
            "target": "es6",
            "outDir": "./dist",
            "baseUrl": "./lib"
        },
        "include": [
            "lib/**/*.ts"
        ],
        "exclude": [
            "node_modules"
        ]
    }

Edit running scripts in package.json

    "scripts": {
        "build": "tsc",
        "dev": "ts-node ./lib/server.ts",        
        "start": "nodemon ./dist/server.js",
        "prod": "npm run build && npm run start"
    }
