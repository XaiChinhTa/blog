Rest Parameters

Trong quá trình làm việc với JavaScript, không ít lần bạn gặp những hàm có số lượng tham số tùy ý như Math.max(arg1, arg2, ..., argN) hoặc Object.assign(dest, src1, ..., srcN). Những tham số trên được gọi là Rest Parameters Vậy làm thế nào để define một function có tham số như thế ?

Thực ra, trong JavaScript, chúng ta có thể gọi function với số lượng tham số tùy ý mà không gây ra lỗi.

    function sum(a, b) {
        return a + b;
    }
      
    sum(1, 2, 3, 4, 5);

Như ví dụ trên, chúng ta có thể truyền đến 5 tham số nhưng hàm sum chỉ nhận và dùng 2 tham số đầu tiên.

Để define Rest Parameters, chỉ cần đặt dấu ... trước tên tham số.

    function sumAll(...args) { // args is the name for the array
        let sum = 0;
      
        for (let arg of args) sum += arg;
      
        return sum;
    }

Trong trường hợp có nhiều tham số thì nên khai báo Rest Parameters sau cùng.

    function showName(firstName, lastName, ...titles) {
        alert( firstName + ' ' + lastName ); // Julius Caesar
      
        // the rest go into titles array
        // i.e. titles = ["Consul", "Imperator"]
        alert( titles[0] ); // Consul
        alert( titles[1] ); // Imperator
        alert( titles.length ); // 2
    }
      
    showName("Julius", "Caesar", "Consul", "Imperator");

Spread Syntax

Chúng ta đã khai báo thành công Rest Parameters. Nhưng lại phát sinh một vấn đề khác là chúng ta không thể truyền một array vào Rest Parameters.

    let arr = [3, 5, 1];

    alert( Math.max(arr) ); // NaN

Vậy làm thế nào để truyền array vào Rest Parameters?

Thay vì phải lặp qua từng phần tử của mảng và truyền vào Rest Parameters thì JavaScript đã hỗ trợ chúng ta một cách ngắn gọn hơn.

    let arr = [3, 5, 1];

    alert( Math.max(...arr) ); // 5 (spread turns array into a list of arguments)
