What is a Module Bundler ?

Module Bundler là một công cụ dùng để đóng gói Javascript hoặc CSS từ nhiều file khác nhau (bao gồm Dependencies) thành một file duy nhất.

Thông thường, Module Bundler sẽ bắt đầu với một entry file(như index.js, style.css, ...). Từ đó, Module Bundler sẽ đóng gói tất cả code cần thiết vào entry file đó.


Why we need Module Bundler ?

Mọi browser luôn giới hạn số lượng kết nối đồng thời tới một domain cũng như giới hạn kết nối đồng thời tổng thể. Vì vậy, việc đóng gói thành một file duy nhất giúp tiết kiệm kết nối tới server nhờ đó ta có thể tối ưu performance khi load trang.

Khi đóng gói thành một file duy nhất cũng giúp việc cache Javascript, CSS trở nên hiệu quả hơn. (Need to clear how browser cache Js and CSS to determine exactly effect of Bunding)

Module Bundler cũng có thể giúp minify và loại bỏ những đoạn code dư thừa không dùng đến để giảm thiểu dung lượng file.
