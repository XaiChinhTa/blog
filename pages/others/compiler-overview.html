What is compiler ?

Theo định nghĩa, compiler là một chương trình giúp chuyển đổi code được viết thành mã máy.

Để giảm thiểu lượng code phải viết cũng như tăng tính trừu tượng của code, người ta cho ra đời nhiều kỹ thuật (ngôn ngữ) khác nhau.

Nhưng để code có thể chạy được, code cũng phải được chuyển về ngôn ngữ nền tảng.
