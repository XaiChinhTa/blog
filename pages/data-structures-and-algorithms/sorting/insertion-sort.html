Idea   :    

The algorithm maintains 2 subarrays in a given array
       The subarray which already sorted
       The subarray which unsorted
In every iteration, the element from the unsorted is picked and moved to the correct position in sorted array.

Worst-case Performance   :     O(n2)
Best-case Performance   :     O(n)
Average Performance   :     O(n2)
Aulixiary Space   :     O(n)
In place sort   :     Yes, it does not require extra space

Implementation

    A = [64, 25, 12, 22, 11] 
        
    # Traverse through 1 to len(arr) 
    for i in range(1, len(arr)): 
  
        key = arr[i] 
  
        # Move elements of arr[0..i-1], that are 
        # greater than key, to one position ahead 
        # of their current position 
        j = i-1
        while j >= 0 and key < arr[j] : 
                arr[j + 1] = arr[j] 
                j -= 1
        arr[j + 1] = key
