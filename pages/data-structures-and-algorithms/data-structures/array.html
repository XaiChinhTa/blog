Array is a collection of items stored at contiguous memory locations.
Array is a kind of Static Data Structure . Therefore the size of array is fixed and must be provided before storing data.
Memory of an array is allocated at compile time.

Accessing Time   :     O(1)
Search Time   :     O(n)
Insertion Time   :     O(n)
Deletion Time   :     O(n)
Memory Allocation   :     Memory is allocated at compile time
