Stack is an abstract data type that serves as a collection of elements with the principal operation Last In, First Out.
It can be implemented by using both array and linked list.
We alsway can remove recursion with stack.

Accessing Time   :     O(n)
Search Time   :     O(n)
Insertion Time   :     O(1)
Deletion Time   :     O(1)
Memory Allocation   :     Memory is allocated at compile time
