Queue is an abstract data type that serves as a collection of elements and supports First In, First Out operations.
It can be implemented by using both array and linked list.

Accessing Time   :     O(n)
Search Time   :     O(n)
Insertion Time   :     O(1)
Deletion Time   :     O(1)
Memory Allocation   :     Memory is allocated at compile time

Priority Queue

Priority Queue is an extension of Queue with following properties

Every item has a priority associated with it.

An high priority element must be dequeued before lower one.

If two elements have the same priority, they are served according to their order in the queue.

A priority queue supports insert, get_highest_priority and delete_highest_priority operations.

Priority Queue is usually implemented by Heap because Heap provides better performance than Array and Linked List

Deque

Deque is called Double Ended Queue or Head Tail Linked List

Deque is an abstract data type that generalizes a Queue that allows to insert and delete at both ends.

Circular Queue

Circular Queue is a Queue and the last position is connected back to the first position to make a circle

Circular Queue is usually used in Memory Management, Traffic System and CPU Scheduling

Circular Queue supports operations

front   :     The first element from queue
rear   :     The last element from queue
check_full   :     The circular queue full when (rare == queue_size - 1 && front == 0) || (rare == front - 1)
enqueue   :    

Insert an element in the queue
         Step 1 : Check the queue is full or not
         Step 2 : Set position of next rare rare = (rare + 1) % queue_size
         Step 3 : Set data into the rare

dequeue   :    

Delete an element in the queue
         If the queue has only one element : front = rare = -1
         If the queue has many elements :     front = (front + 1) % queue_size
