Breadth First Search is usually used to find the shortest path between 2 nodes in undirected or directed, unweighted graphs.

Because BFS visits each node by the graph level, so the first visit of any node is alsway the shortest path from the start node.

BFS ofen has 4 elements

graph   :    

The list of edges in the graph
visited   :    

The list of visited nodes.
Because graph may contain cycles, we need to save the visited node for checking

path   :    

The list for tracking path. From this list, we can trace the shortest path between 2 nodes.
queue   :    

When we go to a node, we push its children to the queue.
The algorithm stops when the queue is empty 

    def BFS(graph, root):
        path = {}
        queue = []
        visited = {}
            
        queue.append(root)
        visited[root] = True
        
        while len(queue):
            node = queue.pop(0)
            
            if node in graph:
                for child in graph[node]:
                    if child not in visited:
                        path[child] = node
                        queue.append(child)
                        visited[child] = True
                    
        return path
