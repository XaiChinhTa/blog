Linked list is a kind of Linear Data Structure like Array.
Each element (node) of the list contains two items - the data and a reference to the next node.
Linked List has dynamic size. The node can be added or removed in the list.

Accessing Time   :     O(n)
Search Time   :     O(n)
Insertion Time   :     O(1)
Deletion Time   :     O(1)
Size   :     Dynamic Size
Memory Allocation   :     Memory is allocated at run time

Singly Linked List
         In this type of linked list, every node stores reference of next node in the list and the last node reference as NULL.

Doubly Linked List
         In Doubly Linked List, every node store references of next and previous nodes. With this structure, we can traverse in both directions.
         For deletion, we don't need to have explicit access to previous node.

Circular Linked List
         Circular Linked List can be Singly or Doubly Linked List, but no node reference to NULL. The nodes reference to form a circle.
         In this type of Linked List, every node can be the starting node.
